Marauder

Introducción al Mundo Maruader

Bienvenido al fascinante universo del Marauder, una herramienta de vanguardia basada en ESP32 que revoluciona el campo de la ciberseguridad. Este dispositivo multifacético permite realizar tanto ataques ofensivos como defensivos en redes WiFi y Bluetooth, rivalizando con dispositivos populares como el Flipper Zero.

Creado por el visionario JustCallMeKoko, el proyecto Marauder democratiza el acceso a herramientas de pentesting avanzadas, permitiendo a entusiastas y profesionales construir su propio dispositivo a un costo accesible.

El ESP32 Marauder: Un Arsenal Digital

El Marauder no es simplemente una herramienta; es un ecosistema completo de ciberseguridad en la palma de tu mano. Con capacidades que incluyen escaneo de redes, desconexión de dispositivos, captura de handshakes WiFi y análisis de dispositivos Bluetooth cercanos, el Marauder se posiciona como un aliado indispensable para cualquier profesional de la seguridad informática.

Arsenal del Hacker: Materiales Necesarios

La construcción de tu propio Marauder es un viaje emocionante. Aquí te presentamos los componentes esenciales para dar vida a esta poderosa herramienta:

Componentes Básicos:

  • ESP32: El cerebro de la operación
  • Pantalla Touch TFT LCD módulo ILI9341 2.4 Pulgadas (Se recomienda 2.8 pulgadas para mayor comodidad)
  • Jumpers: Para interconexiones vitales

Actualización a Versión Portátil:

  • TP4056: Gestor de carga de batería
  • Mini DC-DC boost module: Para estabilidad de voltaje
  • Batería LiPo 3.7v, +1500mAh: Para operaciones extendidas
  • Switch: Control de encendido/apagado
  • Módulo GPS (Opcional): Para funcionalidades avanzadas de geolocalización

Infusión de Inteligencia: Flasheando el ESP32

Para dar vida a tu Marauder, es crucial cargar el firmware correcto. Te presentamos dos métodos probados:

1. Método Web: La Vía Rápida

Ventaja Principal: Simplicidad y Rapidez

  1. Navega al Web Uploader de Spacehuhn: https://esp.huhn.me
  2. Conecta tu ESP32 al puerto USB de tu computadora
  3. Selecciona el puerto COM correspondiente en la interfaz web
  4. Descarga los archivos necesarios del repositorio GitHub de koko
  5. Sube los archivos en el orden correcto y comienza el proceso de flasheo

Consejo: Si encuentras problemas con una versión específica del firmware, no dudes en experimentar con versiones alternativas para encontrar la más compatible con tu hardware.

2. Método Arduino IDE: Para los Puristas

Desafío: Requiere más conocimientos técnicos, pero ofrece mayor control
