Lab 1: Introducción a Docker

Aprende los conceptos básicos y crea tu primer contenedor

Objetivos del Laboratorio

  • Entender los conceptos básicos de Docker
  • Instalar Docker en tu sistema
  • Crear y ejecutar tu primer contenedor Docker
  • Aprender comandos básicos de Docker

Prerrequisitos

Antes de comenzar, asegúrate de tener:

  • Un sistema operativo compatible (Windows, macOS, o Linux)
  • Permisos de administrador en tu sistema
  • Conexión a Internet

Paso 1: Instalación de Docker

Sigue las instrucciones de instalación según tu sistema operativo:

  • Windows
  • macOS
  • Ubuntu

Paso 2: Verificar la instalación

Abre una terminal y ejecuta el siguiente comando para verificar que Docker se ha instalado correctamente:

docker --version

Deberías ver la versión de Docker instalada en tu sistema.

Paso 3: Ejecutar tu primer contenedor

Vamos a ejecutar un contenedor "Hello World" para comprobar que todo funciona correctamente:

docker run hello-world

Este comando descargará una imagen de prueba y la ejecutará en un contenedor. Deberías ver un mensaje de bienvenida.

Paso 4: Comandos básicos de Docker

Aquí tienes algunos comandos útiles para empezar a trabajar con Docker:

  • docker images: Lista todas las imágenes descargadas
  • docker ps: Muestra los contenedores en ejecución
  • docker ps -a: Muestra todos los contenedores (incluyendo los detenidos)
  • docker pull [imagen]: Descarga una imagen del Docker Hub
  • docker stop [contenedor]: Detiene un contenedor en ejecución
  • docker rm [contenedor]: Elimina un contenedor detenido

Reto Final

Ahora que has aprendido los conceptos básicos, intenta lo siguiente:

  1. Descarga la imagen oficial de Nginx usando docker pull nginx
  2. Ejecuta un contenedor de Nginx y mapea el puerto 80 del contenedor al puerto 8080 de tu máquina local
  3. Accede a http://localhost:8080 en tu navegador para ver la página de bienvenida de Nginx

Pista: El comando para ejecutar el contenedor de Nginx debería ser algo como:

docker run -d -p 8080:80 nginx

Conclusión

¡Felicidades! Has completado el Lab 1 de introducción a Docker. Ahora tienes los conocimientos básicos para empezar a trabajar con contenedores. En los próximos labs, profundizaremos en temas más avanzados como la creación de imágenes personalizadas, el uso de Docker Compose y la gestión de redes en Docker.
